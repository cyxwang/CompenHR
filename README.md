# CompenHR
## Introduction:
This is PyTorch implementation of CompenHR.

## Usage
   1. Clone this repo:
  
     git clone https://github.com/cyxwang/CompenHR
     
     cd CompenHR/src/python
     
   2. Start visdom by typing:
   3. 
     visdom

   4. Run train_CompenHR.py to produce results:
   5. 
     python train_DeProCams.py
